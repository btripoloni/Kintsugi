Importa arquivos ou diretórios do sistema de arquivos local.

**Parâmetros:**
- `path: string`: Caminho para o arquivo ou diretório, relativo à raiz do modlist.

Quando essa source é usada, o arquivo que esta em "path" é copiado para dentro da pasta onde vai ficar o resultado da compilação, na raiz, cabe a outras partes do processo de build cuidar desse arquivo.
