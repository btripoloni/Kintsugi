# Compilador
O trabalho de um compilador é, ao receber o caminho de uma receita ele usar as instruções contidas nela para formar uma build.

O compilador é de certa forma "burro" ele executa o que é passado para ele de forma sincronica apenas garantindo que ele esteja correto.
Um compilador não sabe o que é skyrim, minecraft ou outro jogo, ele sabe que tem que baixar arquivos, mover arquivos, escrever arquivos, criar links tanto duros quanto simbolicos.
Em suma executar o que le for pedido.

## O que é uma build
Uma build é o resultado de uma compilação completa, tanto a receita fornecida quanto suas dependencias.
Uma explicação mais completa em ./builds.md
